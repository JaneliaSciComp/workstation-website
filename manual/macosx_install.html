Installing Workstation on Mac OS

These instructions are specifically for Mac OS. If you are using another operating system, you can find instructions here.

Step 1: Remove Existing Workstation Installation, if necessary

  1. Click on your "Applications" icon.
  2. Find the "Janelia Workstation" application.
  3. Drag it to the Trash.
  4. If you have older installations of the Workstation, you can trash them all.

Step 2: Install Java 8

  1. Download the Oracle JDK 8.
  2. Run the installer and follow the wizard to install.
  3. If you already have Java 8 installed, the installer will let you know.

Step 3: Install Janelia Workstation

  1. Download the Janelia Workstation Installer.
  2. Run the installer and follow the wizard to install.
  3. The default options are fine, just press Next until it starts installing.
