Installing Workstation on Windows

These instructions are specifically for Windows. If you are using another operating system, you can find instructions here.

Windows 10 Security Notes

  1. If you are using Windows 10, Windows Defender SmartScreen may prevent downloaded installers from running.
  2. If you see this screen, click the "More info" link.
  1. Then click the "Run anyway" button.
  1. Click "Yes" if Windows User Account Control asks to allow the app to make changes to your device.

Step 1: Remove Existing Workstation Installation, if necessary

  1. Click the Windows Start button
  2. Select "Control Panel"
  3. Click on "Uninstall a program"
  1. Right-click the Janelia Workstation, and select "Uninstall/Change"
  2. This is also a good opportunity to uninstall older versions of Java, if you have any. Unless you specifically need them (e.g. for running Matlab), it's a good idea to remove older versions of Java, because they may contain security holes.
  1. When uninstalling the Workstation, you can check the "Remove directory" checkbox to remove the old configuration folder. This folder does not store your important preferences, only things like Window positioning and application logs.

Step 2: Install Java 8

  1. Download Oracle JDK 8.
  2. Run the downloaded executable and follow the wizard to install the JDK.
  3. When the installer completes it will automatically run the JRE installer. Allow this to install as well.

Step 3: Install Janelia Workstation

  1. Download the Janelia Workstation Installer.
  2. Run the installer and follow the wizard to install.
  3. The default options are fine, just press Next until it starts installing.

Trouble installing?

  1. In some cases, after uninstalling and attempting to install again, you can get an error which says "The specified installation folder is not empty."
  2. Open Windows Explorer
  3. Navigate to "C:\Program Files\"
  4. Delete the "JaneliaWorkstation" folder
  5. Try installing again
  6. If you still have problems, please contact us.

Windows Firewall

  1. When you first run the Janelia Workstation, the Windows Defender Firewall may block some features of the application.
  2. Under most circumstances, "Domain networks" or "Private networks" will already be checked. If so, click "Allow access".
  3. If only "Public networks" is available, click "Cancel", then contact your local workstation administrator regarding how the workstation servers are set up and what the correct firewall setting is for you.
