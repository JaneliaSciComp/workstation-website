Installing the Janelia Workstation on Linux

These instructions are specifically for Linux. If you are using another operating system, you can find instructions here.

Step 1: Remove Existing Workstation Installation, if necessary

  1. Execute uninstall.sh from your Workstation install directory
  1. Follow the wizard promots to uninstall the program
  2. When uninstalling the Workstation, you can check the "Remove directory" checkbox to remove the old configuration folder. This folder does not store your important preferences, only things like Window positioning and application logs.

Step 2: Install Java 8

  1. Scientific Computing Systems maintains a shared copy of JDK 8 at /misc/local/jdk1.8.0_71, which you can likely use on your system.
  2. To install JDK 8 on your own Redhat-based system, just run "yum install" on the RPM.
  3. If you have multiple versions installed, you can make JDK 8 active by using the "alternatives --config java" command.
  4. Alternatively, you can modify the JaneliaWorkstation shortcut to add a --jdkhome parameter pointing to your JDK 8 installation.

Step 3: Install Janelia Workstation

  1. Download the Janelia Workstation Installer.
  1. Run the installer and follow the wizard to install.
  2. The default options are fine, just press Next until it starts installing.
