Remote Access

The Workstation client works over VPN. However, if you prefer not to install the Workstation client on your machine, there are some other ways that you can access Workstation data.

To see your imagery in a web browser, the Webstation allows browsing of the same secondary data as the Workstation.

Another alternative to installing the Workstation is to run the Workstation client on Janelia's compute cluster and stream compressed video to a client machine on Janelia's network or remotely via VPN. You can start a NoMachine session to connect to the cluster and easily run the Workstation by selecting it from the Janelia application menu.
