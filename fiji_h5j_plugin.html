H5J Loader for Fiji

We are currently working to integrate the 12-bit H5J Loader into the main Fiji distribution. In the meantime, you can use the manual installation as follows:

  • Download the installer
  • Start Fiji
  • Use Plugins → Install Plugin and select the installer you downloaded
  • Restart Fiji
  • Run the installer plugin (Plugins → H5J Installer)
  • Restart Fiji
  • You should now be able able to open 12-bit H5J images
